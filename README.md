# 💬 WaveChat

WaveChat is a modern real-time chat application built using Node.js, Express, and Socket.io.

## 🚀 Features
- Real-time messaging
- Public channels & private chats
- Typing indicators
- Online/offline status
- User search & discovery
- Chat management (mute, archive)

## 🛠 Tech Stack
- Node.js + Express
- Socket.io
- HTML, CSS, JavaScript

## 📦 Installation

```bash
cd BackEnd
npm install
node server.js
